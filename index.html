Solution

Site para consulta de algoritmos para diversos problemas envolvendo programação.


Sobre o site

Solution é um site do GitHub Pages que tem como objetivo principal exibir as soluções desenvolvidas para diversos problemas do URI Online Judge Academic , além disso, é possível encontrar e consultas as funções criadas nas soluções dos problemas de forma organizada no site. Em geral, essas funções podem ser reutilizadas em outros problemas ou compor a solução de problemas mais complexos
